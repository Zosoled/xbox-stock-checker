how to use

  1. go to desired xbox product page (sample links below)
    • why? same-origin policy i.e. browser security
  2. hit f12 to open dev tools
  3. switch to "console" tab in dev tools
  4. copy code from xbox-stock-checker.js
  5. paste into console and press enter

code runs in background while browser tab remains open
